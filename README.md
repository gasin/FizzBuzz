## Description
Try to maintain scalability and maintainability and efficiency

## Usage

### make
```bash
$ make
```
### run test
```bash
$ make test
```

### input file format

```
s e m
num_1 name_1
num_2 name_2
...
num_m name_m
```

### output file format
```
name_of_{s}
name_of_{s+1}
...
name_of_{e}
```
